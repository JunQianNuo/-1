# 问题三初步回答与建模表述：写作设计

## 目标与范围

新增 `问题分析/星链系统-文献驱动版/18-问题三初步回答与建模表述.md`。文档面向论文写作，参照问题一的“初步回答与建模表述”结构，集中呈现问题三的模型、求解结论、检验和适用边界；不复制参数化模型、算法设计与运行报告中的完整推导和伪代码。

全文只采用仓库中已经存在的模型、运行数据、检验结果和文献证据。每一项物理或工程假设均以“科学依据”标注对应已有文献或已有模型文档；数值结论则标出结果文件来源。

## 内容结构

1. **结论摘要**：给出固定构型下的饱和搜索结论：归档搜索返回 `S*=1680`，对应 `P30_all=0.8564212703336913`，饱和窗口 `[1680,1880]` 的增量为 `0.47%`；论文中采用保守表述 `S_sat≲1680、平台约0.85`。
2. **时变星间网络模型**：说明时间离散快照、可见性接入、四近邻 ISL、边权时延和按时隙最短路。只保留论文需要的数学表达，并链接至 `21-问题三参数化求解模型`。
3. **问题二—问题三联合反推与饱和判停**：先以问题二覆盖约束筛选候选构型，再以 `P30_all` 的扩展窗口增量和每百星增量决定是否饱和。说明这是“最小且后续改善不显著”的搜索准则，不是硬阈值达标。
4. **数值结果与检验**：分两层陈述。
   - 归档饱和搜索的原始结论使用 `joint_summary.json`；
   - 正式敏感性与 Bootstrap 检验使用独立、显式的标准网格，得到 S=1540、1680、1760 的均值约为 0.8110、0.8216、0.8297。该网格的抽样规模与旧搜索相同，但其通信坐标不能逐点复现归档运行，故绝不将两者当作同一数值实验。
5. **可直接入论文的表述与边界**：给出一段克制结论，明确“加星的边际收益变小”仅适用于固定倾角、初相位、路由和时延口径；未声称全局最优或将全部未达标 OD 对归因于单一的几何下界。

## 非目标

- 不重跑饱和搜索或改动算法、参数和结果；
- 不把独立检验结果伪装成归档搜索的逐点复现；
- 不添加仓库以外的定量数据、经验阈值或参考文献；
- 不把 `S*=1680` 写成整个设计空间的全局最优星数。

## 验收标准

- 文件名、frontmatter 和写作层次与问题一对应文档保持一致；
- 文中所有关键数值均可追溯到上述结果文件；
- 科学依据和“本项目数值结果”在表达上清晰分开；
- 明确写出归档搜索与独立标准网格检验的口径差异和结论边界。
